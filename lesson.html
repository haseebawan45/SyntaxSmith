Variables and Data Types

Introduction to Variables

Variables are containers for storing data values in JavaScript. Unlike other programming languages, JavaScript doesn't make you declare a variable's type - a variable can hold a value of any data type.

In modern JavaScript, we have three ways to declare variables:

  • let - used for variables whose values will change
  • const - used for variables whose values will remain constant
  • var - older way to declare variables (generally avoided in modern code)

Declaring Variables

Here's how to declare variables in JavaScript:

JavaScript 
// Using let for variables that will change
let age = 25;
let name = "John";

// Using const for variables that won't change
const PI = 3.14159;
const MAX_SIZE = 100;

// Changing a let variable's value
age = 26;

// This would cause an error: 
// PI = 3.14; // Cannot reassign a constant

Always use const by default, and only use let when you know the variable's value will change.

JavaScript Data Types

JavaScript has eight basic data types:

String

Represents textual data enclosed in quotes:

let name = "John";
let message = 'Hello';

Number

Represents both integers and floating-point numbers:

let age = 25;
let price = 19.99;

Boolean

Represents a logical value:

let isActive = true;
let hasPermission = false;

Null

Represents a deliberate non-value:

let user = null;

Undefined

Represents an unassigned value:

let message;
// Value is undefined

Object

Complex data structure:

let person = {
name: "John",
age: 25
};

Symbol

Unique and immutable primitive value:

let id = Symbol("id");

BigInt

For integers of arbitrary length:

let bigNumber = 1234567890123456789012345n;

Type Conversion

JavaScript is a dynamically typed language, which means you don't need to specify the data type of a variable when you declare it. This also means that data types can be automatically converted when needed.

JavaScript 
// String to Number
let strNumber = "42";
let number = Number(strNumber);  // Explicit conversion
console.log(number);             // 42

// Number to String
let value = 42;
let strValue = String(value);    // Explicit conversion
console.log(strValue);           // "42"

// Boolean conversion
console.log(Boolean(1));         // true
console.log(Boolean(0));         // false
console.log(Boolean(""));        // false
console.log(Boolean("hello"));   // true

Beware of automatic type conversion in JavaScript, especially when using the == operator. Use the === operator for strict equality checks without type conversion.

Summary

  • Variables are declared using let, const, or var.
  • Use const by default, and let when the variable value will change.
  • JavaScript has eight primitive data types: String, Number, Boolean, Null, Undefined, Object, Symbol, and BigInt.
  • JavaScript performs automatic type conversion when needed.
  • Use explicit conversion methods like Number(), String(), and Boolean() for clarity.
